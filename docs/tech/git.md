# Git

## 常忘指令

update from remote branch

```bash title="update remote branch"
$ git fetch origin discover:discover && git checkout discover
# or
$ git fetch origin aRemoteBranch
$ git checkout aLocalBranch
$ git merge origin/aRemoteBranch
```

空目錄

加一個 `.gitkeep` 檔案
